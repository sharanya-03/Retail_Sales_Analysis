# Retail_Sales_Analysis
# Retail Sales Data Analysis and Sales Prediction

## Project Overview

This project demonstrates an end-to-end data science workflow using a real-world retail sales dataset. The objective is to analyze sales patterns, visualize important business insights, and build a machine learning model to predict sales.

The project covers data preprocessing, exploratory data analysis (EDA), visualization, feature engineering, model training, evaluation, and conclusions.

---

## Objectives

* Analyze retail sales data.
* Clean and preprocess the dataset.
* Explore relationships between different business variables.
* Visualize sales trends and distributions.
* Build a machine learning model to predict sales.
* Evaluate model performance using standard regression metrics.

---

## Dataset

**Dataset Name:** Superstore Sales Dataset

The dataset contains information about retail orders, including:

* Order Date
* Category
* Segment
* Region
* Ship Mode
* Quantity
* Discount
* Sales
* Profit
* Customer Details

## Project Workflow

### 1. Data Collection

* Load the Superstore sales dataset.

### 2. Data Cleaning

* Check missing values.
* Remove duplicate records.
* Verify data types.

### 3. Exploratory Data Analysis (EDA)

* Summary statistics.
* Sales distribution.
* Category-wise sales analysis.
* Regional sales analysis.
* Monthly sales trend.

### 4. Feature Engineering

* Encode categorical variables.
* Select relevant features for prediction.

### 5. Machine Learning Model

* Random Forest Regressor

### 6. Model Evaluation

Performance is evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score
